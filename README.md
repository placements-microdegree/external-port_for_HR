# External Portal for HR

A React-based web application that helps HR teams view and identify top candidates for recruitment and placement opportunities.

## Overview

This project provides an external-facing portal designed to streamline the candidate evaluation process for HR departments. It displays candidate information in an organized and user-friendly interface, helping HR professionals quickly identify the best candidates.

**Live Demo:** [https://talent.microdegree.work](https://talent.microdegree.work)
